# Kepler Exoplanet Habitability Explorer

An interactive Streamlit web application designed to analyze and explore potentially habitable exoplanets using NASA's Kepler dataset.

## ✨ Features
* Filter and explore exoplanets based on Earth-like properties.
* Interactive data visualizations for planetary locations and habitability metrics.
* Clean and structured data preprocessing.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Streamlit, Pandas, Jupyter Notebook
